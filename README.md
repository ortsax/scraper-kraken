## Orsta.py

A simple REST API for miscellaneous utility tasks.

### Installation

To get started with the project, ensure you have Python 3.14 and `uv` installed.

1. Clone the repository:

```bash
git clone https://github.com/realastrox11/Orsta.py
cd Orsta.py

```

2. Install dependencies:

```bash
uv sync

```

3. Run the application:

```bash
uv run python main.py

```

Alternatively, you can run the project using Docker:

1. Build and start the container:

```bash
docker-compose up --build

```
